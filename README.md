# 一次性打开多个标签页

一个简单的一次性打开多个链接至标签页的Chrome扩展

## 预览

![](image/screenshots.png)

## 功能介绍

输入多个链接，方便的一次性在新标签页中全部打开

## 如何使用

输入需要打开的链接，一行一条
